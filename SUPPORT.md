# Support

Use [GitHub Issues](https://github.com/gearboxlogic/localhold/issues) for bug
reports, installation problems, and focused feature requests. Include the
LocalHold version, operating system, storage backend, transport, relevant
configuration with secrets removed, and reproducible logs.

Use [GitHub Discussions](https://github.com/gearboxlogic/localhold/discussions)
for usage questions and design discussion when Discussions are enabled.

The project does not currently provide a paid support SLA.
