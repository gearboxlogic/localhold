# Plan of the Hold — explorations

Working references from the 2026-07 redesign of the secondary mark. Not part
of the shipped brand kit; the canonical mark is `../bastion-plan.svg`.

Comparison gallery (rounds 1–3):
https://claude.ai/code/artifact/85120d68-fb0a-4e49-9bd9-35f2e854516e

The SVGs are generated by the checked-in scripts (`fortgen.py`, `round2.py`,
`round3.py`; run from this directory, they write into subdirectories). All
geometry is parametric: curtain radius `rc`, tip radius `rt`, neck fraction
`f`, shoulder depth `s`, plus ward radius and gate/causeway gap widths.

Naming: round 1 explored trace shapes (a–f); round 2 iterated the pentagon
(b0–b3) and moat (f0–f1); round 3 added the counterscarp causeway (b2c, b2o).
